Readme file for CCSS Summer 2023 practice sessions. 
Download package from public GitHub repository, link below.

Package contains following folder structure:
Code - Contains subfolders 'R', 'Stata', and 'Python'. Here is where you will find sample programming files. Coding files use relative paths. Start with no applications open. To replicate results double click on programming file you want to run. This will automatically set directory to correct location. Then submit commands. If you want to re-run, close out of all applications, then re-open by double clicking on programming file to submit commands. 
Data - Contains sample data used for manipulation and analysis exercises
Output - Contains output files when generated. Log files tracking output and code. Output tables.
